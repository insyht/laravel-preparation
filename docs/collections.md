# Collections
- Maken van een collectie op basis van een array (https://laravel.com/docs/6.x/collections#creating-collections)
- Maak een collection macro (extra functionaliteit aan een collection toevoegen) (https://laravel.com/docs/6.x/collections#extending-collections)
- Gebruik de verschillende methods van een collection (https://laravel.com/docs/6.x/collections#method-listing)
    - Alles teruggeven (https://laravel.com/docs/6.x/collections#method-all)
    - Splits een collectie op in stukken met x aantal items per stuk (https://laravel.com/docs/6.x/collections#method-chunk) -> bijvoorbeeld handig als je een grid hebt in een view
    - Splits een collectie op in x aantal stukken (https://laravel.com/docs/6.x/collections#method-split)
    - Voeg meerdere arrays samen in 1 collectie (https://laravel.com/docs/6.x/collections#method-collapse)
    - Voeg meerdere arrays samen waarbij de eerste voor de keys zorgt en de tweede voor de values (https://laravel.com/docs/6.x/collections#method-combine)
    - Maak een kopie van de huidige collectie in een nieuwe variabele (https://laravel.com/docs/6.x/collections#method-collect)
    - Voeg een nieuwe array toe aan het eind van een bestaande collectie (https://laravel.com/docs/6.x/collections#method-concat)
    - Voeg een array toe aan een collectie. Als een key al bestaat, dan het origineel behouden (https://laravel.com/docs/6.x/collections#method-union)
    - Voeg een item toe aan het begin van een collectie (https://laravel.com/docs/6.x/collections#method-prepend)
    - Voeg een item toe aan het eind van een collectie (https://laravel.com/docs/6.x/collections#method-push)
    - Voeg een item toe aan een collectie waarbij je zowel de key als de value kan meegeven (https://laravel.com/docs/6.x/collections#method-put)
    - Check of een bepaald item bestaat in de collectie (https://laravel.com/docs/6.x/collections#method-contains, https://laravel.com/docs/6.x/collections#method-containsstrict)
        - Doe dit ook met een closure
    - Check of een bepaalde key bestaat in een collectie (https://laravel.com/docs/6.x/collections#method-has)
    - Tel hoeveel items er in een collection zitten (https://laravel.com/docs/6.x/collections#method-count)
    - Tel het aantal items, gegroepeerd (https://laravel.com/docs/6.x/collections#method-countBy)
    - Geef een matrix terug met alle mogelijke combinaties tussen de huidige collectiewaardes en een of meer arrays (https://laravel.com/docs/6.x/collections#method-crossjoin)
    - Geef een verschil tussen een collectie en een andere collectie of array (https://laravel.com/docs/6.x/collections#method-diff, https://laravel.com/docs/6.x/collections#method-diffassoc, https://laravel.com/docs/6.x/collections#method-diffkeys)
    - Geef duplicates terug, inclusief hoe vaak ze erin zitten (https://laravel.com/docs/6.x/collections#method-duplicates, https://laravel.com/docs/6.x/collections#method-duplicatesstrict)
    - Geef unieke waardes terug (vergeet niet om ze te sorteren met $collection->values()) (https://laravel.com/docs/6.x/collections#method-unique, https://laravel.com/docs/6.x/collections#method-uniquestrict)
    - Callback over alle collectie items uitvoeren zonder dat je ze wil aanpassen (return in de callback niet verplicht) (https://laravel.com/docs/6.x/collections#method-each, https://laravel.com/docs/6.x/collections#method-eachspread)
    - Pas waardes aan in een collectie met een callback (return WEL verplicht) in de collectie zelf (https://laravel.com/docs/6.x/collections#method-map, https://www.youtube.com/watch?v=1M8Latarclc)
    - Pas waardes aan in een collectie met een callback (return WEL verplicht), en geef de nieuwe collectie terug (https://laravel.com/docs/6.x/collections#method-map, https://www.youtube.com/watch?v=1M8Latarclc)
    - Pak de waardes van een collectie en voer die aan een class constructor. Maak daar een nieuwe collectie van (https://laravel.com/docs/6.x/collections#method-mapinto)
    - Je hebt een collectie van meerdere arrays (bijv door chunk) en je wil elk van die arrays (die uit bijv 2 waardes bestaan) voeren aan een callback functie (https://laravel.com/docs/6.x/collections#method-mapspread)
    - Maak een nieuwe collectie van een bestaande, waarbij je values uit de oude gebruikt als keys voor de nieuwe (https://laravel.com/docs/6.x/collections#method-maptogroups)
    - Maak een nieuwe collectie van een bestaande, waarbij je meerdere values uit de oude combineert tot nieuwe items (https://laravel.com/docs/6.x/collections#method-mapwithkeys)
    - Check of elk item aan een voorwaarde voldoet (https://laravel.com/docs/6.x/collections#method-every)

    - Filter met callback (als false, dan in de collectie houden) (https://laravel.com/docs/6.x/collections#method-reject)
    - Geef het eerste item terug die voldoet aan een bepaalde voorwaarde (https://laravel.com/docs/6.x/collections#method-first, https://laravel.com/docs/6.x/collections#method-first-where)

    - Maak van een multi-dimensionele collectie een platte collectie (https://laravel.com/docs/6.x/collections#method-flatten)
    - Draai de keys en values om (https://laravel.com/docs/6.x/collections#method-flip)
    - Haal een key uit een collectie (https://laravel.com/docs/6.x/collections#method-forget)
    - Pagineer een collectie (https://laravel.com/docs/6.x/collections#method-forpage)
    - Geef de waarde van een gegeven key (https://laravel.com/docs/6.x/collections#method-get)
    - Groepeer een collectie (https://laravel.com/docs/6.x/collections#method-groupby)
    - Implodeer een collectie op basis van een key (https://laravel.com/docs/6.x/collections#method-implode)

    - Haal alle items uit een collectie die niet in een gegeven array zitten (https://laravel.com/docs/6.x/collections#method-intersect, https://laravel.com/docs/6.x/collections#method-intersectbykeys)
    - Check of een collectie wel of niet leeg is (https://laravel.com/docs/6.x/collections#method-isempty, https://laravel.com/docs/6.x/collections#method-isnotempty)
